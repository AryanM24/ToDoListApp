# ToDoListApp
This intuitive to-do list application is designed to streamline task management and enhance productivity. With features like task categorization, deadline tracking, and the ability to prioritize tasks with a simple star, it keeps you focused on what matters most. Upcoming deadlines are automatically highlighted at the top, ensuring you never miss an important task. Built with Svelte, this app is perfect for students, professionals, and anyone looking to stay organized and productive!
